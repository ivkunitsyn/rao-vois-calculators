# Калькулятор ставок РАО для вещателей радиоканалов

Статус: реализован (MVP).

Реализовано:
- отдельный Python-движок расчёта `backend/rao_radio.py`;
- ветка Генерального соглашения РАР–РАО–ВОИС;
- ветка общего Положения РАО;
- расчёт ставки по тематикам/долям/хронометражу;
- расчёт минимальной суммы (в т.ч. коэффициенты по часам и интернет-надбавки);
- итог `% + минимум` и режим фиксированной суммы.

В текущем релизе ВОИС-радио не подключён (следующий этап).
