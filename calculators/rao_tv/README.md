# Калькулятор ставок РАО для вещателей телеканалов

## Что внутри

- `backend/api.py` — основной FastAPI backend.
- `backend/rao.py` — расчётный движок по Положению РАО для ТВ.
- `backend/calculator.py` — Python entrypoint калькулятора.
- `data/` — рабочие данные для ТВ-калькуляторов:
  - таблица РКН;
  - slim-версия таблицы;
  - SQLite-индекс;
  - индекс ИНН→наименование;
  - таблица переменных ставок.

## Роль в системе

Этот модуль является базовым для ТВ-калькуляторов:

- РАО ТВ работает напрямую на нём;
- ВОИС ТВ использует тот же расчётный пайплайн, но с отдельным набором ставок и параметров.
