# Калькулятор ставок ВОИС для вещателей телеканалов

Реализация использует общий ТВ-движок расчёта (`calculators/rao_tv/backend/rao.py`) с отдельным набором параметров ВОИС:

- `data/Переменные из ставок ВОИС.xlsx`

Во фронтенде выбор выполняется через меню калькуляторов. Для backend передаётся `calc_type=vois_tv`.
