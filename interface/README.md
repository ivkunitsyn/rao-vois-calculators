# Frontend

Frontend проекта находится в одном месте:

- `interface/index.html`

Это общий shell для всех калькуляторов. Он:

- показывает верхнее меню калькуляторов;
- переключает пользователя между 6 режимами;
- хранит состояние раздельно по калькуляторам;
- вызывает backend API для нужного калькулятора;
- не содержит расчётной бизнес-логики — расчёты остаются в Python backend-модулях.

Если меняется только интерфейс, правки вносятся здесь.
Если меняется только логика расчёта, правки вносятся в `calculators/*/backend/`.
